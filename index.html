Cómo usar este sitio

1

Elige un modelo

Selecciona Largo Plazo o Corto Plazo y luego una pestaña de modelo.

2

Set 1 = escenario base

Ajusta los deslizadores del Set 1 al escenario que quieras como base.

3

Copia y cambia algo

Haz clic en "Copiar 1→2" y luego cambia un solo parámetro en el Set 2.

4

Compara y exporta

Lee las tablas y gráficos, luego exporta como imagen, CSV o Excel.
